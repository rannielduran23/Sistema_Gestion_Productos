# Sistema_Gestion_Productos

Introducción:
Este proyecto consiste en el desarrollo de un sistema básico para la gestión de productos, implementado en el lenguaje de programación C#. Su propósito principal es facilitar el registro, la búsqueda, la eliminación y el listado de productos dentro de un almacén, a través de un menú interactivo en consola.

El ejercicio está orientado a poner en práctica conceptos fundamentales de la programación orientada a objetos, tales como la definición de clases, la aplicación de encapsulamiento y el establecimiento de relaciones entre objetos. Asimismo, se aborda el manejo de estructuras de datos para el almacenamiento de objetos, contrastando el uso de arreglos (arrays) con colecciones genéricas como List.
El sistema desarrollado se caracteriza por ser sencillo, funcional y estar bien estructurado, lo que permite comprender de manera clara cómo organizar un programa que administre información de forma eficiente y ordenada.
